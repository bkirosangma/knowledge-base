# Music SVG Renderer Catalog

Generated by `kb_svg.py --type <renderer>`. All renderers share common flags:

| Flag | Default | Purpose |
|------|---------|---------|
| `--out <path>` | stdout | Write SVG to file instead of stdout |
| `--width N` | 600 | Canvas width in px |
| `--height N` | 400 | Canvas height in px |
| `--theme light\|dark` | light | Color scheme |
| `--title "..."` | (none) | Title text drawn at top |
| `--tradition <tag>` | (none) | Tint accents using tradition badge color |
| `--midi-out <path>` | (none) | Emit MIDI alongside SVG (Phase 4, where supported) |
| `--play` | false | Play generated MIDI (Phase 4) |

Tradition tags: `western` `jazz` `hindustani` `carnatic` `maqam` `persian` `gamelan` `west-african` `east-asian` `latin` `folk`

## 1. `staff` — Western Staff Notation

Renders a single-line Western staff with optional notes, clef, time signature, and accidentals.

### Flags

| Flag | Default | Values / Notes |
|------|---------|---------------|
| `--notes "..."` | (empty) | Space-separated `<pitch>-<dur>` tokens, e.g. `C4-q D4-q E4-h`. Duration codes: `w` whole, `h` half, `q` quarter, `e` eighth, `s` sixteenth |
| `--clef <name>` | treble | `treble` `bass` `alto` `tenor` |
| `--time-sig N/N` | 4/4 | Time signature, e.g. `3/4` `6/8` |

### Example

```bash
python3 kb_svg.py \
  --type staff \
  --notes "C4-q D4-q E4-q F4-q G4-h G4-h" \
  --clef treble \
  --time-sig 4/4 \
  --title "C major scale" \
  --out staff-c-major.svg
```

### Output shape

Five horizontal staff lines across the canvas. Clef symbol at left. Time signature digits. Each note drawn as a filled/open note head with stem; sharps/flats rendered as accidental glyphs.

## 2. `fretboard` — String Instrument Fretboard

Renders an N-string fretboard (12 frets) with tuning labels, standard dot markers, and optional fret highlights.

### Flags

| Flag | Default | Values / Notes |
|------|---------|---------------|
| `--strings N` | 6 | Number of strings |
| `--tuning <str>` | EADGBE | String tuning, high to low. Either a 6-char string (`EADGBE`) or comma-separated (`E,A,D,G,B,E`). Use comma form for multi-char names (`E,A,D,G,B,E` or `D,A,D,G,B,E`) |
| `--notes "..."` | (empty) | Comma-separated fret numbers to highlight (applied to all strings) |

### Example

```bash
python3 kb_svg.py \
  --type fretboard \
  --strings 6 \
  --tuning EADGBE \
  --notes "0,3,5,7" \
  --tradition western \
  --title "E minor pentatonic root position" \
  --out fretboard-em-pentatonic.svg
```

### Output shape

Horizontal strings with tuning labels at left. Vertical fret lines (nut bolded). Standard dot markers at frets 3, 5, 7, 9, and double-dot at 12. Highlighted frets drawn as accent-colored circles on every string.

## 3. `circle-of-fifths` — Western Circle of Fifths

Renders the 12-key circle with major keys on the outer ring and relative minors on the inner ring. Optionally highlights one or more keys.

### Flags

| Flag | Default | Values / Notes |
|------|---------|---------------|
| `--highlight "..."` | (empty) | Comma-separated key names to accent, e.g. `"C,G,D"` for a cluster |

### Example

```bash
python3 kb_svg.py \
  --type circle-of-fifths \
  --highlight "F,C,G" \
  --title "Dorian modal region" \
  --out circle-dorian.svg
```

### Output shape

Two concentric circles with 12 spokes. Outer ring: major key labels. Inner ring: relative minor labels. Highlighted keys drawn in accent color. Grid spokes in muted color.

## 4. `raga-clock` — Hindustani / Carnatic Raga Clock

Renders a 12-pitch-class or 22-shruti wheel with arohana (ascending scale) highlighted for a named raga.

### Flags

| Flag | Default | Values / Notes |
|------|---------|---------------|
| `--raga <name>` | (empty) | Raga name from built-in set: `Yaman` `Bhairav` `Bhairavi` `Yaman Kalyan` `Marwa` `Bilawal` `Khamaj` `Kafi` `Asavari` `Todi` `Purvi`. Falls back to no-highlight for unknown ragas |
| `--shrutis N` | 12 | `12` for semitone grid, `22` for shruti grid (24-element approximation) |
| `--show-vadi` | false | (stub) Mark vadi and samvadi notes when implemented |

### Example

```bash
python3 kb_svg.py \
  --type raga-clock \
  --raga Yaman \
  --tradition hindustani \
  --title "Raga Yaman — arohana" \
  --out raga-yaman.svg
```

### Output shape

Single circle. 12 (or 22) evenly spaced dots around the circumference, labeled with solfege swara names (`Sa re Re ga Ga Ma Ma# Pa dha Dha ni Ni`) or shruti numbers. Notes in the raga are accent-colored and connected by lines. Raga name centered.

## 5. `maqam-jins` — Arabic / Turkish Maqam Jins

Renders a named jins (tetrachord) as a horizontal cents axis with quartertone-aware pitch labels.

### Flags

| Flag | Default | Values / Notes |
|------|---------|---------------|
| `--jins <name>` | (empty) | Jins from built-in set: `Rast` `Bayati` `Hijaz` `Saba` `Kurd` `Ajam` `Nahawand` `Sikah`. Falls back to `[0,200,400,500]` generic tetrachord |
| `--root <letter>` | D | Root note letter (`C` `D` `E` `F` `G` `A` `B`) |
| `--show-cents` | false | Print cent values below each tick mark |

### Example

```bash
python3 kb_svg.py \
  --type maqam-jins \
  --jins Hijaz \
  --root D \
  --show-cents \
  --tradition maqam \
  --title "Jins Hijaz on D" \
  --out jins-hijaz-d.svg
```

### Output shape

Horizontal axis spanning 0–max_cents. Tick marks every 100 cents. Accent-colored circles at each interval step, labeled with diatonic pitch names (quartertone steps suffixed `+` or `d`). Jins name as title above axis.

## 6. `gamelan-cipher` — Javanese / Balinese Cipher Notation

Renders gamelan cipher notation in slendro (5-note, digits 1–2–3–5–6) or pelog (7-note, digits 1–7) tuning.

### Flags

| Flag | Default | Values / Notes |
|------|---------|---------------|
| `--tuning slendro\|pelog` | slendro | Determines valid digit set; digits outside the tuning are muted |
| `--pattern "..."` | (empty) | Space-separated cipher digits, e.g. `"1 2 3 5 6 5 3 2"` |

### Example

```bash
python3 kb_svg.py \
  --type gamelan-cipher \
  --tuning slendro \
  --pattern "1 2 3 5 6 5 3 2 1" \
  --tradition gamelan \
  --title "Pelog / Slendro demo" \
  --out gamelan-slendro-demo.svg
```

### Output shape

Evenly spaced cells across the canvas. Each cell shows one cipher digit in large bold type. Digits valid in the chosen tuning are accent-colored; out-of-tuning digits are muted. Tuning label above. Grid separators between cells.

## 7. `chord-box` — Chord Diagram (Guitar / Ukulele / Bass / Mandolin)

Renders a chord box (box diagram) for a named chord. Fingering given as fret numbers per string.

### Flags

| Flag | Default | Values / Notes |
|------|---------|---------------|
| `--instrument guitar\|uke\|bass\|mandolin` | guitar | Determines number of strings (6 / 4 / 4 / 8) |
| `--chord "..."` | (empty) | Chord name displayed above the box, e.g. `"Dm7"` |
| `--fingering "..."` | (empty) | Space-separated tokens, one per string (low → high). `x` = muted, `0` = open, digit = fret number, e.g. `"x x 0 2 1 1"` |

### Example

```bash
python3 kb_svg.py \
  --type chord-box \
  --instrument guitar \
  --chord "Dm7" \
  --fingering "x x 0 2 1 1" \
  --tradition jazz \
  --title "Dm7 — open position" \
  --out chord-dm7.svg
```

### Output shape

Grid with N vertical string lines and 5 horizontal fret lines. Nut bolded at top. `x` marks above muted strings, open circles above open strings, filled accent circles at fingered frets. Chord name above the box.
